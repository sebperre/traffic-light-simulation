# Setup

```bash
sudo apt-get update
sudo apt-get install -y gdal-bin libgdal-dev python3
```

Add to .bashrc:
```bash
export CPLUS_INCLUDE_PATH=/usr/include/gdal
export C_INCLUDE_PATH=/usr/include/gdal
```

Then
```bash
source ~/.bashrc
```
